# esphome-projects
A collection of my ESPHome configurations and notes for ESP32- and ESP8266-based devices used with Home Assistant.
This repository contains ESPHome YAML files, device setups, and troubleshooting notes for sensors, controllers, and other smart home hardware. The goal is to keep my device configs organized, reusable, and easy to update.

## What’s in this repo

- ESPHome YAML configuration files
- Device-specific notes
- Pin mappings and wiring references
- Troubleshooting examples
- Test configs for new hardware

## Example Devices

This repository may include projects such as:

- Motion sensors
- Presence sensors
- Temperature and humidity sensors
- Relay modules
- LED controllers
- ESP32 and ESP32-C3 development boards
